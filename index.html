Features

  1. Seperates JavaScript from Page
  2. Totally self contained (JQuery independant)
  3. Very fast (not creating dom nodes on the fly)
  4. Reconfigurable on each request
  5. Editable cells and validation
  6. Uses minimul HTML tags (table related tags, no div's, spans)
  7. Cross browser compatible
  8. Small and compact
  9. Fully resizable without resize code
  10. Readable source code
